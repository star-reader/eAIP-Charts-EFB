# eAIP 航图查看器 (EFB)

一个现代化的电子飞行包（Electronic Flight Bag）应用，专为航空从业者设计，提供便捷的航图查看和管理功能。

## 🚀 项目简介

eAIP 航图查看器是基于 Vue 3 + Tauri 构建的跨平台桌面应用，支持查看和管理各类航空图表，包括机场图、航路图、SUP、AIC、NOTAM 等文档。

### 主要功能

- **机场航图管理** - 支持机场图、进离场程序、仪表进近程序等
- **航路图查看** - 提供航路图和区域图的浏览功能
- **官方文档** - 集成 SUP、AIC、NOTAM 等官方航空文档
- **PDF 阅读器** - 内置高性能 PDF 查看器，支持缩放、旋转、平移等操作
- **响应式设计** - 完美适配桌面端和移动端设备

## 🛠 技术架构

### 前端技术栈
- **Vue 3** - 渐进式 JavaScript 框架
- **TypeScript** - 类型安全的 JavaScript 超集
- **Vite** - 现代化的前端构建工具
- **Vue Router** - 官方路由管理器
- **SCSS** - CSS 预处理器

### 桌面端技术
- **Tauri** - 轻量级桌面应用框架
- **Rust** - 高性能系统编程语言

### 核心依赖
- **vue-pdf-embed** - PDF 文档渲染
- **naive-ui** - Vue 3 UI 组件库
- **pubsub-js** - 组件间通信

## 🔧 开发环境

### 环境要求
- Node.js 16+
- pnpm (推荐) 或 npm
- Rust (用于 Tauri 构建)

### 安装依赖
```bash
pnpm install
```

### 开发模式
```bash
pnpm run dev
```

### 构建应用
```bash
# Web 版本
pnpm run build

# 桌面应用
pnpm run tauri build
```

## 🎯 使用说明

1. **选择类别** - 在主导航中选择要查看的航图类别
2. **选择具体项目** - 在子导航中选择具体的航图类型或机场
3. **浏览航图** - 在弹出的选择面板中选择要查看的航图
4. **PDF 操作** - 使用右侧控制面板进行缩放、旋转等操作

### PDF 查看器功能
- **缩放控制** - 25% 到 600% 的缩放范围
- **手势操作** - 支持鼠标滚轮、触控板手势
- **大文件优化** - 自动检测大文件并提供优化选项
- **页面导航** - 多页文档的页面切换
